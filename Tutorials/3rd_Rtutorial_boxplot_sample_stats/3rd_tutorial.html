Introduction

We are going to consider some fastfood data set from openintro.org. I’ll show how to use the mosaic package to calculate summary statistics and gf_boxplot to make box plots. Your assignment is on the last page.

  1. read about the data set. (see openintro.org)

  2. Load the openintro library for the data and the mosaic library for the functions by typing them into the console.

library(openintro)
library(mosaic)

Sample Statistics

  1. We want to look at the fastfood data. We saw several ways to do this in the first R tutorial. Type one into the chunk below.
glimpse(fastfood)
Quiz
  1. Let’s find some sample statistics. favstats() from the mosaic library will spit out the normal summary statistics. You can see the all the statistics you need document for more functions we can use here.
favstats(~calories,data=fastfood)
  1. Notice the calories are all over the place from 20 to 2430, this is caused by the different types of foods from salads to burgers to fries. We should edit the formula to break the food into restaurants.
favstats(calories~restaurant,data=fastfood)
Quiz

Boxplots

We can view these five number summaries as boxplots. Copy the chunk from above and change favstats() to gf_boxplot().

gf_boxplot(calories~restaurant,data=fastfood)

Note the outliers. We will learn how to remove those much later in the course. The coef defaults to 1.5 but we can change our coef for the outlier whiskers by adding coef =3 as an argument to gf_boxplot(). Try it.

gf_boxplot(calories~restaurant,data=fastfood)
gf_boxplot(calories~restaurant,data=fastfood,coef = 3)

To see all the things to change type ?gf_boxplot() into a chunk. Try to give a good title, change the color, and improve the axes labels.
